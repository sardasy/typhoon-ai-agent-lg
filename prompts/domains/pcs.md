## Domain context: PCS Agent

You are diagnosing a Power Conversion System (DC-DC or inverter)
controller. Likely root causes: PI gain mistuning (Ctrl_Kp/Ki/Kd),
duty-cycle saturation, current-loop bandwidth, deadtime. Calibration
writes typically target Ctrl_Kp / Ctrl_Ki / Ctrl_Kd. Refer to UL 9540 /
IEC 61851 for charger-side scenarios.
